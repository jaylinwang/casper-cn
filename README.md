# casper-cn

由 Ghost 默认主题改造而来，适配中文博客的风格。

原项目地址：[Casper](https://github.com/TryGhost/Casper)

# 开发

- 环境准备
  - [Node](https://nodejs.org/)
  - [Yarn](https://yarnpkg.com/)
  - [Gulp](https://gulpjs.com)

- 开发

```bash
$ yarn install
$ yarn dev
```
